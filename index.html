All Deep Learning Security
Data Privacy using Deep Learning

Implementing a deep learning model to protect sensitive data from information leakage. This model uses a min-mix optimization process to learn statistical features of real data to produce synthesized data with similar statistical features as real data.

Application Security in IDE(ASIDE)33

Software vulnerabilities originating from insecure code are one of the leading causes of security problems people face today. We propose a new approach, called interactive static analysis, to integrate the detection and mitigation of security vulnerabilities into the context of development.

Attack Evaluation

This project aims to evaluate vulnerable web pages against XSS attack vectors in unit tsting environment such as JUnit. The project uses headless browser component such as HtmlUnit sn JWebUnit to simulate the real browses in the unit testing framework. The attack vectors contain paylaods that can be dectected by the evaluation module depicting vulnerabilities in the code

Grammar based Attack Generator

This project aims to generate XSS attack vectors for different contexts of HTML, CSS, JavaScript. This project employes the context-switching mechanism of the HTMl languages to generate attack vectors that can be used in different context. The grammar of this attack generator is manually defined and can be cusimized to add new context-swtiching features of Html family languages.
